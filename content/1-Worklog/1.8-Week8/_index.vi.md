* Tìm hiểu cách thiết lập hồ lưu trữ dữ liệu tập trung (Data Lake) và phân tích dữ liệu kinh doanh (Business Intelligence).

### Kết quả đạt được tuần 8:

* Hiểu cách tổ chức hồ dữ liệu chuẩn hóa, phân quyền truy cập chi tiết bằng AWS Lake Formation.
* Thiết lập thành công dashboard trực quan, báo cáo KPI kinh doanh bằng Amazon QuickSight.
* Nắm vững cách dùng Athena truy vấn trực tiếp trên file log/CSV lưu tại S3 bằng ngôn ngữ SQL.
* Hiểu cách điều chỉnh tham số, tối ưu chỉ mục (index) và sao lưu cơ sở dữ liệu PostgreSQL.
* Thực hiện trọn vẹn vòng đời Machine Learning (Data Prep, Train, Tune, Deploy) trên môi trường Amazon SageMaker.
